# ユーティリティスクリプト

便利なスクリプト集です。

## スクリプト一覧

| スクリプト | 説明 |
|-----------|------|
| check_dependencies.py | 依存関係のチェック |
| test_voicevox.py | VOICEVOXの接続テスト |
| test_vtube_studio.py | VTube Studioの接続テスト |

## 使い方

```bash
# 依存関係チェック
python scripts/check_dependencies.py

# VOICEVOXテスト
python scripts/test_voicevox.py

# VTube Studioテスト
python scripts/test_vtube_studio.py
```
