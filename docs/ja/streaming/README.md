## 概要

AITuber配信の基本構成：

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│ VTube       │     │ OBS         │     │ 配信        │
│ Studio      │────▶│ Studio      │────▶│ プラット    │
│ (アバター)   │     │ (配信ソフト) │     │ フォーム    │
└─────────────┘     └─────────────┘     └─────────────┘
       ▲                   ▲
       │                   │
┌──────┴──────┐     ┌──────┴──────┐
│ AITuber     │     │ 仮想        │
│ Starter Kit │────▶│ オーディオ   │
│ (音声出力)   │     │ (VB-Cable)  │
└─────────────┘     └─────────────┘
```

## OBS Studioの設定

### 1. シーンの構成

推奨シーン構成：

```
📁 シーン
├── 🎬 配信中
│   ├── VTube Studio（ゲームキャプチャ）
│   ├── 背景画像
│   └── オーバーレイ
├── 🎬 開始前
│   └── 待機画面
└── 🎬 終了
    └── エンディング画面
```

### 2. VTube Studioの取り込み

1. ソースを追加 → 「ゲームキャプチャ」
2. モード: 「特定のウィンドウをキャプチャ」
3. ウィンドウ: VTube Studioを選択
4. 「透過を許可」にチェック

### 3. 音声設定

**VB-Cableを使用する場合:**

1. AITuber Starter Kitの音声出力を「VB-Cable Input」に設定
2. OBSで音声入力ソースを追加
3. デバイス: 「VB-Cable Output」を選択

**直接出力の場合:**

1. デスクトップ音声をOBSで取り込み
2. 他のアプリ音声と混ざる点に注意

### 4. 出力設定

**推奨設定（配信）:**

| 項目 | 推奨値 |
|------|--------|
| エンコーダ | NVENC（GPU）/ x264（CPU） |
| ビットレート | 4500-6000 kbps |
| キーフレーム間隔 | 2秒 |
| プリセット | Quality |

**推奨設定（映像）:**

| 項目 | 推奨値 |
|------|--------|
| 基本解像度 | 1920×1080 |
| 出力解像度 | 1920×1080 |
| FPS | 30 |

## プラットフォーム別設定

### YouTube Live

詳細は [youtube-live.md](youtube-live.md) を参照

**クイック設定:**
1. YouTube Studioで配信を作成
2. ストリームキーをコピー
3. OBS → 設定 → 配信 → YouTube - RTMPS
4. ストリームキーを貼り付け

### Twitch

詳細は [twitch-setup.md](twitch-setup.md) を参照

**クイック設定:**
1. Twitchダッシュボードでストリームキーを取得
2. OBS → 設定 → 配信 → Twitch
3. アカウントを接続
