## インストール

1. [OBS公式サイト](https://obsproject.com/)からダウンロード
2. インストーラーを実行
3. 初回起動時の自動設定ウィザードを完了

## 基本設定

### 一般設定

設定 → 一般

- **言語**: 日本語
- **テーマ**: お好みで
- **システムトレイ**: 有効推奨

### 映像設定

設定 → 映像

| 項目 | 推奨値 | 説明 |
|------|--------|------|
| 基本解像度 | 1920×1080 | キャンバスサイズ |
| 出力解像度 | 1920×1080 | 配信解像度 |
| 縮小フィルタ | Lanczos | 高品質 |
| FPS | 30 | 配信安定性重視 |

### 出力設定

設定 → 出力 → 配信タブ

**シンプルモード:**

| 項目 | 推奨値 |
|------|--------|
| ビットレート | 4500 kbps |
| エンコーダ | ハードウェア（利用可能な場合） |
| 音声ビットレート | 160 kbps |

**詳細モード:**

| 項目 | 推奨値 |
|------|--------|
| エンコーダ | NVENC H.264 または x264 |
| レート制御 | CBR |
| ビットレート | 4500-6000 kbps |
| キーフレーム間隔 | 2秒 |
| プリセット | Quality |
| プロファイル | high |

## シーン設定

### 推奨シーン構成

```
📁 シーンコレクション: AITuber配信
│
├── 🎬 待機画面
│   └── 配信開始前の待機画像
│
├── 🎬 メイン配信
│   ├── VTube Studio（アバター）
│   ├── 背景
│   └── オーバーレイ
│
├── 🎬 雑談
│   ├── VTube Studio
│   └── シンプル背景
│
└── 🎬 終了画面
    └── エンディング画像
```

### シーンの作成

1. シーンパネルで右クリック
2. 「追加」を選択
3. シーン名を入力

### シーン切り替え

- **手動**: シーンをクリック
- **ホットキー**: 設定 → ホットキーで割り当て
- **スタジオモード**: プレビュー → 配信の切り替え

## ソース設定

### VTube Studioの取り込み

**方法1: ゲームキャプチャ（推奨）**

1. ソースを追加 → ゲームキャプチャ
2. モード: 「特定のウィンドウをキャプチャ」
3. ウィンドウ: `[VTube Studio.exe]: VTube Studio`
4. ✅ 透過を許可
5. ✅ アンチチート互換性フックを使用

**方法2: ウィンドウキャプチャ**

1. ソースを追加 → ウィンドウキャプチャ
2. ウィンドウ: VTube Studioを選択
3. キャプチャ方法: Windows Graphics Capture

**方法3: Spout2（高度）**

1. VTube StudioでSpout2出力を有効化
2. OBSにSpout2プラグインをインストール
3. Spout2キャプチャソースを追加

### 背景設定

1. ソースを追加 → 画像
2. 背景画像を選択
3. VTube Studioソースより下に配置

### オーバーレイ

1. ソースを追加 → 画像
2. フレームやロゴを追加
3. 最前面に配置

## 音声設定

### 音声デバイス

設定 → 音声

| 項目 | 設定 |
|------|------|
| サンプルレート | 48kHz |
| チャンネル | ステレオ |
| デスクトップ音声 | 既定 または VB-Cable Output |
| マイク音声 | 無効（AITuberは使用しない） |

### 音声ミキサー

**AITuber音声のみの場合:**
- デスクトップ音声: 有効
- その他: ミュート

**VB-Cable使用の場合:**
- VB-Cable Output: 有効
- その他: ミュート

### 音声フィルタ

デスクトップ音声 → フィルタ

推奨フィルタ：
1. **ノイズ抑制**: RNNoise
2. **コンプレッサー**: 音量の均一化

## 配信設定

### YouTube Live

設定 → 配信

1. サービス: YouTube - RTMPS
2. サーバー: Primary YouTube ingest server
3. ストリームキー: YouTube Studioからコピー

### Twitch

設定 → 配信

1. サービス: Twitch
2. 「アカウント接続」をクリック
3. Twitchにログイン

### カスタムRTMP

設定 → 配信

1. サービス: カスタム
2. サーバー: RTMPサーバーURL
3. ストリームキー: 配信キー

## トラブルシューティング

### 映像がカクカクする

1. 出力解像度を下げる
2. FPSを30に下げる
3. エンコーダをハードウェアに変更

### 音声が途切れる

1. 音声ビットレートを下げる
2. サンプルレートを確認
3. オーディオドライバを更新

### VTube Studioが取り込めない

1. OBSを管理者権限で実行
2. ゲームキャプチャの設定を確認
3. VTube Studioを再起動
