## イラスト制作ツール

### 比較表

| ツール | 価格 | プラットフォーム | 難易度 | おすすめ度 |
|--------|------|-----------------|--------|-----------|
| CLIP STUDIO PAINT | 月額480円〜 / 買い切り5,000円〜 | Win/Mac/iPad | ★★★☆☆ | ★★★★★ |
| Procreate | 2,000円（買い切り） | iPad専用 | ★★☆☆☆ | ★★★★☆ |
| Photoshop | 月額2,728円〜 | Win/Mac/iPad | ★★★★☆ | ★★★★☆ |
| Krita | 無料 | Win/Mac/Linux | ★★★☆☆ | ★★★☆☆ |
| MediBang Paint | 無料 | Win/Mac/iOS/Android | ★★☆☆☆ | ★★★☆☆ |

### 詳細

#### CLIP STUDIO PAINT（推奨）

**メリット:**
- Live2D用パーツ分けに最適
- PSD出力が高品質
- アニメーション機能あり
- 日本語サポートが充実

**デメリット:**
- 有料（ただし買い切りあり）
- 機能が多く最初は戸惑う

#### Procreate

**メリット:**
- 直感的な操作
- 買い切りで安価
- 豊富なブラシ

**デメリット:**
- iPad専用
- PSD出力の互換性に注意

#### Krita

**メリット:**
- 完全無料・オープンソース
- PSD対応
- 高機能

**デメリット:**
- 動作がやや重い
- 日本語情報が少ない

## Live2D制作ツール

### 比較表

| ツール | 価格 | 用途 | 難易度 |
|--------|------|------|--------|
| Live2D Cubism Editor | 無料〜月額2,200円 | モデル制作 | ★★★★☆ |
| VTube Studio | 無料（一部有料） | 表示・トラッキング | ★★☆☆☆ |
| nizima LIVE | 無料 | 表示・トラッキング | ★★☆☆☆ |
| FaceRig | 買い切り1,480円 | 表示・トラッキング | ★★☆☆☆ |

### 詳細

#### Live2D Cubism Editor

| エディション | 価格 | 機能 |
|-------------|------|------|
| FREE | 無料 | 基本機能、パーツ数制限あり |
| PRO | 月額2,200円 | 全機能、商用利用可 |

**選び方:**
- 趣味・個人利用 → FREE版で十分
- 商用・本格的 → PRO版推奨

#### VTube Studio（推奨）

**メリット:**
- 高精度なフェイストラッキング
- iPhoneのFace IDに対応
- カスタムパラメータ対応
- API連携可能（本プロジェクトで使用）

**デメリット:**
- 高機能ゆえに設定項目が多い

## 3Dモデル制作ツール

### 比較表

| ツール | 価格 | 難易度 | VRM出力 | おすすめ度 |
|--------|------|--------|---------|-----------|
| VRoid Studio | 無料 | ★☆☆☆☆ | ✓ | ★★★★★（初心者） |
| Blender | 無料 | ★★★★★ | アドオン要 | ★★★★★（中〜上級） |
| Maya | 月額36,300円 | ★★★★★ | アドオン要 | ★★★☆☆ |
| Metasequoia | 買い切り5,500円〜 | ★★★☆☆ | プラグイン要 | ★★★☆☆ |

### 詳細

#### VRoid Studio（初心者推奨）

**メリット:**
- 完全無料
- 直感的なUI
- VRM直接出力
- 髪の毛を手描きで作成可能

**デメリット:**
- デザインの自由度に限界
- VRoid感が出やすい

#### Blender（中〜上級者推奨）

**メリット:**
- 完全無料・オープンソース
- 無限の自由度
- 豊富なチュートリアル
- VRMアドオンで出力可能

**デメリット:**
- 学習コストが高い
- 3Dモデリングの知識が必要

## アバター表示ソフト

### 比較表

| ソフト | Live2D | VRM | 価格 | API |
|--------|--------|-----|------|-----|
| VTube Studio | ✓ | ✓ | 無料（PRO 2,500円） | ✓ |
| nizima LIVE | ✓ | ✗ | 無料 | ✗ |
| 3tene | ✗ | ✓ | 無料〜有料 | ✗ |
| VMagicMirror | ✗ | ✓ | 無料 | ✗ |
| VDRAW | ✓ | ✗ | 無料 | ✗ |

### 本プロジェクトでの推奨

**VTube Studio を推奨**

理由：
1. Live2D/VRM両対応
2. API連携でプログラムから制御可能
3. 高精度トラッキング
4. 活発な開発・サポート

## 選定フローチャート

### 2D（Live2D）の場合

```
Q: 予算は？
├── 無料で始めたい
│   ├── イラスト: Krita / MediBang
│   ├── Live2D: Cubism FREE
│   └── 表示: VTube Studio（無料版）
│
└── 有料でも良い
    ├── イラスト: CLIP STUDIO PAINT（推奨）
    ├── Live2D: Cubism PRO
    └── 表示: VTube Studio PRO
```

### 3D（VRM）の場合

```
Q: 3Dモデリング経験は？
├── 初心者
│   ├── モデル: VRoid Studio
│   └── 表示: VTube Studio / 3tene
│
└── 経験あり
    ├── モデル: Blender + VRMアドオン
    └── 表示: VTube Studio / VMagicMirror
```
