## イラスト制作

### 推奨キャンバスサイズ

| 用途 | サイズ | 解像度 |
|------|--------|--------|
| 配信用（推奨） | 4000×4000px | 72dpi |
| 高品質 | 6000×6000px | 72dpi |

### 注意点

- **正面向き**のイラストを基本とする
- 左右対称を意識する（後の作業が楽になる）
- パーツごとに分けやすい構図にする
- 線画と塗りは別レイヤーにしない（1パーツ1レイヤー）

## パーツ分け

### 基本パーツ構成

```
📁 キャラクター
├── 📁 顔
│   ├── 顔輪郭
│   ├── 📁 左目
│   │   ├── まぶた上
│   │   ├── まぶた下
│   │   ├── 白目
│   │   ├── 瞳
│   │   └── ハイライト
│   ├── 📁 右目
│   │   └── （同上）
│   ├── 左眉
│   ├── 右眉
│   ├── 鼻
│   ├── 📁 口
│   │   ├── 上唇
│   │   ├── 下唇
│   │   ├── 口内
│   │   └── 歯
│   └── 頬（照れ）
├── 📁 髪
│   ├── 前髪
│   ├── 横髪（左）
│   ├── 横髪（右）
│   └── 後ろ髪
├── 📁 体
│   ├── 首
│   ├── 体
│   ├── 左腕
│   └── 右腕
└── 背景（オプション）
```

### パーツ分けのコツ

1. **動かしたい部分を別レイヤーに**
   - 瞬きさせたい → まぶたを分ける
   - 口を動かしたい → 唇を分ける

2. **隠れる部分も描く**
   - 前髪の下の額
   - 閉じた目の下の部分
   - 口を開けた時の口内

3. **PSD形式で保存**
   - レイヤー名は英数字推奨
   - グループ化して整理

## Live2D Cubismでの作業

### 1. PSDの読み込み

1. Live2D Cubism Editorを起動
2. ファイル → 新規モデル
3. PSDファイルをドラッグ&ドロップ

### 2. メッシュの作成

各パーツにメッシュを設定：

1. パーツを選択
2. 「メッシュの自動生成」をクリック
3. 変形が必要な部分は細かく設定

**メッシュ設定の目安:**
- 顔輪郭: 細かめ（変形が多い）
- 瞳: 細かめ（回転・拡縮）
- 体: 粗め（あまり動かない）

### 3. デフォーマの設定

**回転デフォーマ**
- 首の回転
- 腕の回転

**ワープデフォーマ**
- 顔の傾き
- 体の動き

### 4. パラメータの作成

AITuberで使用する主要パラメータ：

| パラメータ名 | 用途 | 範囲 |
|------------|------|------|
| ParamAngleX | 顔の左右 | -30〜30 |
| ParamAngleY | 顔の上下 | -30〜30 |
| ParamAngleZ | 顔の傾き | -30〜30 |
| ParamEyeLOpen | 左目開閉 | 0〜1 |
| ParamEyeROpen | 右目開閉 | 0〜1 |
| ParamMouthOpenY | 口の開閉 | 0〜1 |
| ParamMouthForm | 口の形 | -1〜1 |

### 5. モーションの作成（オプション）

表情やリアクション用のモーションを作成：

- 笑顔
- 驚き
- 悲しみ
- 怒り

## VTube Studioへの導入

### 1. ファイルの配置

```
VTube Studio/
└── Live2DModels/
    └── YourModel/
        ├── YourModel.moc3
        ├── YourModel.model3.json
        ├── YourModel.physics3.json（オプション）
        └── textures/
            └── texture_00.png
```

### 2. VTube Studioでの設定

1. VTube Studioを起動
2. 設定アイコン → モデルを選択
3. 作成したモデルを選択

### 3. リップシンク設定

1. 設定 → リップシンク設定
2. 使用するパラメータを選択
   - `ParamMouthOpenY` → 口の開閉
3. 感度を調整

## AITuberとの連携設定

### カスタムパラメータ

AITuber Starter Kitは `AITuberMouthOpen` というカスタムパラメータを作成します。

**VTube Studioでの設定:**

1. 設定 → パラメータ設定
2. `AITuberMouthOpen` を `ParamMouthOpenY` にマッピング
3. 入力範囲: 0〜1
4. 出力範囲: 0〜1

### 表情のホットキー設定

1. 設定 → ホットキー設定
2. 表情ごとにホットキーを作成
3. ホットキーIDをメモ（コードから呼び出し用）

**推奨ホットキー:**
- `happy` - 笑顔
- `sad` - 悲しみ
- `surprised` - 驚き
- `angry` - 怒り
