## VRMとは

VRMは、3Dアバターのためのファイルフォーマットです。

**特徴:**
- プラットフォーム間で互換性がある
- 人型アバターに特化
- 表情や視線の制御に対応
- ライセンス情報を埋め込み可能

## VRoid Studioでの作成

### インストール

1. [VRoid Studio公式サイト](https://vroid.com/studio)からダウンロード
2. インストーラーを実行

### キャラクター作成

#### 1. 新規作成

1. VRoid Studioを起動
2. 「新規作成」をクリック
3. ベースとなる性別を選択

#### 2. 顔の編集

**調整項目:**
- 顔の輪郭
- 目の形・大きさ・位置
- 眉の形
- 鼻の形
- 口の形

#### 3. 髪型の編集

**方法1: プリセット使用**
- 既存の髪型から選択

**方法2: 手描き**
- 「髪型を編集」で自由に描画
- ガイドに沿って髪を配置

#### 4. 体型の編集

- 身長
- 体型（胸・腰・足）
- 肌の色

#### 5. 衣装の設定

- プリセット衣装から選択
- またはカスタムテクスチャを適用

### エクスポート

1. 右上の「書き出し」をクリック
2. VRM設定を確認
   - タイトル
   - 作者名
   - 利用条件
3. 「VRMエクスポート」をクリック

## Blenderでの作成

### 必要なアドオン

1. **VRM Add-on for Blender**
   - [GitHub](https://github.com/saturday06/VRM-Addon-for-Blender)からダウンロード
   - Blenderにインストール

### 基本的な流れ

```
1. キャラクターモデル作成
   ↓
2. リギング（Armature設定）
   ↓
3. ウェイトペイント
   ↓
4. シェイプキー作成（表情）
   ↓
5. VRMエクスポート
```

### リギング

VRMに必要なボーン構成：

```
📁 Armature
├── Hips
│   ├── Spine
│   │   ├── Chest
│   │   │   ├── Neck
│   │   │   │   └── Head
│   │   │   │       ├── LeftEye
│   │   │   │       └── RightEye
│   │   │   ├── LeftShoulder
│   │   │   │   └── LeftUpperArm
│   │   │   │       └── LeftLowerArm
│   │   │   │           └── LeftHand
│   │   │   └── RightShoulder
│   │   │       └── （同様）
│   ├── LeftUpperLeg
│   │   └── LeftLowerLeg
│   │       └── LeftFoot
│   └── RightUpperLeg
│       └── （同様）
```

### シェイプキー（表情）

必須のシェイプキー：

| 名前 | 用途 |
|------|------|
| Neutral | デフォルト |
| A | あ（口） |
| I | い（口） |
| U | う（口） |
| E | え（口） |
| O | お（口） |
| Blink | 瞬き |
| Joy | 喜び |
| Angry | 怒り |
| Sorrow | 悲しみ |
| Fun | 楽しい |

### エクスポート設定

1. ファイル → エクスポート → VRM
2. メタ情報を入力
   - タイトル
   - 作者
   - バージョン
3. 「Export VRM」をクリック

## VTube Studioでの使用

### VRMモデルの読み込み

1. VTube Studioを起動
2. 設定 → モデルを選択
3. 「VRMモデルを追加」をクリック
4. VRMファイルを選択

### リップシンク設定

VRMモデルでは、ブレンドシェイプを使用してリップシンクを行います。

1. 設定 → リップシンク設定
2. 「VRMリップシンク」を有効化
3. 感度を調整

### 表情設定

1. 設定 → 表情設定
2. VRMの表情をホットキーに割り当て

## トラブルシューティング

### モデルが正しく表示されない

- ボーン構造がVRM仕様に準拠しているか確認
- テクスチャが正しく適用されているか確認

### リップシンクが動かない

- ブレンドシェイプが正しく設定されているか確認
- A/I/U/E/Oのシェイプキーが存在するか確認

### パフォーマンスが悪い

- ポリゴン数を削減（目安: 3万ポリゴン以下）
- テクスチャサイズを最適化（2K推奨）
